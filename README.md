# SpendWise 📊

O **SpendWise** é um dashboard de gerenciamento financeiro desenvolvido como atividade avaliativa para a FIAP. O objetivo principal do projeto é consolidar conhecimentos práticos em estruturação semântica de HTML, estilização responsiva com Tailwind CSS e versionamento de código utilizando Git e GitHub.

## 🚀 Tecnologias Utilizadas

* **HTML5:** Estruturação semântica da página utilizando as tags `<nav>`, `<header>`, `<main>` e `<section>`.
* **Tailwind CSS (v4.0):** Framework utilitário utilizado via CLI para toda a estilização, aplicando conceitos de design Mobile-First, grids responsivos e identidade visual profissional.
* **Git & GitHub:** Ferramentas para controle de versão e hospedagem do código-fonte.

## 📱 Funcionalidades e Layout

A tela de Dashboard foi projetada de forma totalmente responsiva, adaptando-se perfeitamente de desktops a dispositivos móveis:
* **Sidebar Navegável:** Barra lateral com links estruturados para a plataforma.
* **Barra Superior (Header):** Contém o título da tela e a identificação do perfil do usuário.
* **Cards de KPI:** Exibição ágil de métricas essenciais como Saldo Total, Gastos Mensais, Investimentos e Alertas Ativos.
* **Transações Recentes:** Uma tabela organizada contendo o histórico financeiro fictício.

## 🛠️ Como Executar o Projeto Localmente

O projeto foi configurado de forma independente para que possa ser aberto diretamente em qualquer máquina:

1. Baixe ou clone este repositório.
2. Navegue até a pasta `dist/`.
3. Dê um duplo clique no arquivo `index.html` para abrir o Dashboard diretamente no seu navegador de preferência.
